pub fn match_sequences_bounded( &mut self, scoring_function: Option<fn(T, T) -> i32>, gap_penalty: Option<i32>, mismatch_penalty: Option<i32>, max_matrix_cells: usize, ) -> Result<(usize, Vec<(Option<usize>, Option<usize>)>), NeedlemanWunschError>

Globally align the sequences, refusing work above max_matrix_cells before allocating the traceback matrix. Scores use two rolling rows, so only traceback storage remains quadratic.
